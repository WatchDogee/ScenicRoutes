Common Issues

Seeing the "Welcome to nginx" page?

This usually means Nginx is not using your application's configuration. Check:

  • Nginx configuration path in the container
  • Document root settings
  • PHP-FPM configuration

Application not loading correctly?

This could be due to:

  • Missing or incorrect .env file
  • Database connection issues
  • Laravel bootstrap errors
